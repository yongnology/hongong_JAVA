02-3 타입 변환

자동 타입 변환(promotion)은 말 그대로 자동으로 타입 변환이 일어나는 것을 의미한다.
자동 타입 변환은 값의 허용 범위가 작은 타입이 허용 범위가 큰 타입으로 저장될 때 발생한다.

byte < short < int < long < double

char 타이브이 경우 int 타입으로 자동 타입 변환되면 유니코드 값이 int 타입에 저장된다.
ex) char charValue = 'A';
	int intValue = charValue;	//65가 저장된다.

※ byte 타입은 char 타입으로 자동 타입 변환될 수 없다.
	왜냐하면 char 타입의 허용범위는 음수를 포함하지 않는데,
	byte 타입은 음수를 포함하기 떄문이다.
